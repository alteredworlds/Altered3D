User Defined Object Properties <<CONTENTS<<

Default behavior can be modified by supplying parameters in MAX's "User Defined Properties" for an Object or Camera. Please check the included sample scenes for examples. Some of these parameters were used to build our SaverGame - 'SpaceGnats' and worked in that context: there may be 'issues' resulting from more general usage - specifically BILLBOARD and ENVIRON.

Description: Usage:
Use 'Separate Specular Color' OpenGL extension for single Object: SPECULAR=T
Disable Lighting for a single Object: NOLIGHT=T
Enable GL Materials for single Object (overide global settings):: MATERIAL=T
Prevent user rotation of Saver Camera: LOCK=T
Pick Object to be rotated by user: MOVE=T
Supply sample interval for animated mesh (overide global setting for single Object): MESHSTEP=n
For Bump Mapped object, specify single light producing the bumps: BUMPLIGHT=name

Billboard Object. About vertical axis, use 0, 1 or full rotation about object center.

 BILLBOARD=0
Skybox - position relative to camera selected on export. Will then be positioned in same way relative to all cameras. ENVIRON=T
