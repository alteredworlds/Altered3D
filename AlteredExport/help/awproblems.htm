(1) Scene doesn't show up - everything is black...

Please make sure that all cameras in the scene have suitable clipping planes set. AlteredWorlds' engine requires that all MAX cameras have the option 'Clip Manually' checked, with suitable values in the 'Near Clip' and 'Far Clip' fields.
In the future, our engine will determine suitable values on a per-frame basis by examining run-time data, but for now these values must be supplied with each camera.
