Launch The AlteredSaver Configuration Dialog

(i) Double click on the AlteredSaver icon:

OR

(ii) Launch the Windows 'Display Properties' dialog- open the 'Control Panel' and double click on the 'Display Properties' icon

Select the 'Screen Saver' tab, look in the drop down list of Screen Savers and select the new entry, 'AlteredSaver'. After a momentary pause, you should see something similar to the dialog to the right.

First, hit the 'Settings' button to chose a suitable configuration for your system. You should see the 'AlteredSaver v3.0 Configuration' dialog below.

Saver Settings

Hit the 'Browse' button near 'Scene to display' to select a new 3D scene to play first.

Check 'Switch to next scene after this number of minutes-' to enable automatic scene-switching. AlteredSaver will switch to the next installed scene every few minutes - enter the desired play time in minutes using the spin control (use the up and down arrows).

Hit Preview Saver to view your selection!

Downloading New Content

When connected to the Internet, press 'Content?'. After a brief delay you'll be show a list of all available new AlteredSaver content.

Select a scene and press 'Download!'. If all goes well, the progress indicator and download speed will be updated as shown below.

 

'Stop' will halt the download - it can be resumed later without loss of data.

When the download finishes, the scene is available for viewing - select it using the 'Saver' page.

HTTP Proxy settings should be taken from Windows.

<Top of Page>

Display Settings

You can either let the saver use your usual screen mode ['Use current screen size and color depth'] or else switch to a mode your video card can accelerate at full-screen. As an example, if you have an older card, you'll only be able to accelerate up to 800x600 in 16bit colour. You may usually keep your computer running at 1024x768 32bit, so in this case you'd want to keep the default options selected above to make sure the saver switches to the correct mode when it runs.

Hit "3D Configuration" and you should see the dialog below.

 

The Altered 3D Configuration dialog allows specification of various different OpenGL settings.

<Top of Page>

If you experience any problems / crahses, please set these values as shown to the right. Some drivers have problems with display lists now and then...!

Sound Settings

We supply you with a short sound loop in WAV form, which we hope you'll like - this will be configured for playback when AlteredSaver is installed. If you wish to turn off all sound playback, please select 'No Sounds'.

'Play Audio CD' invokes your Windows CD Player and plays any Audio CD present in your CDROM drive. You may select the correct CDROM drive using the driver letter list box if your system has more than one CDROM drive installed.

<Top of Page>

'Launch helper to play file' allows you to specify any sound file your system can play based on its file extension. As an example, if you use the Windows Explorer to find your sound file, and you can play the file by double clicking on it, then AlteredSaver should be able to play it. [Please note that AlteredSaver is launching a third party application to play the sound, so AlteredWorlds will not be held responsible for any problems resulting from such use!]

Controls Settings

When playing a SaverGame™, you can swap the effect of the up and down arrow keys to match your preference.

You can select Mouse control for a SaverGame™ launched using the 'Game' page 'Play Now!' button. All screen savers use the key controls and vanish when a mouse movement is detected.

You may also pick the destination folder for any saver screen shots using the 'Browse...' button.

<Top of Page>

Once the options are set to your liking, hit OK to go back to the 'Display Properties' dialog. You should now hit the 'Preview' button to make sure all is well.
