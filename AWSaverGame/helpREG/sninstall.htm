Basic Install Procedure - Registered Version

Run 'AlteredSaver3.exe'. You'll be guided through the installation process, which should be pretty straightforward.

Please make sure you are connected to the internet during the installation process, as AlteredSaver needs to get an unlock code from AlteredWorlds.com. If this presents a problem, please contact us at support@alteredworlds.com for a manual unlock code - see below.

You'll first have to agree to our license agreement.

Next, you must enter your name, address and serial number. The serial number is given in the purchase confirmation email you received when you bought AlteredSaver v3.

This information (and nothing else) is sent to AlteredWorlds and we automatically send back your personal AlteredSaver activation code. If you have difficulties with the auto install, please contact support@alteredworlds.com. Send us the EXACT name, email address and serial number and we'll send you back an unlock code to enter manually. It is usually best to 'cut & paste' this information as capital letters and spaces DO make a difference...!

Pick your destination folder - the value already selected should be just fine.

Select your setup type:

'AlteredSaver v3 - Screen Saver' is already selected and will be fine for most people.
'AlteredSaver v3 - Screen Saver and MAX Exporter' if you would like to try our 3D Studio MAX Exporter plugin, please pick this second option. You'll be asked to specify your 3D Studio MAX Plugin folder, so you'll need to know where 3D Studio MAX is installed! All '3rd Party' plugins like ours are put in a folder named 'plugins' underneath the main 3D Studio MAX install directory; the install program needs to know exactly where this is.

Once all the files have been extracted, the install is done. <Top of Page>

 

Running AlteredSaver v3!

You can start the AlteredSaver configuration dialog in a few different ways:

(i) Click on the new icon on your desktop
(ii) Click on the menu item - Start | Programs | AlteredWorlds | AlteredSaver v3!
(iii) From the Windows 'Display Properties' Dialog, go to the 'Screen Saver' page. Select 'AlteredSaver' from the list of screen savers, then click on the 'Settings' button.

For more help: support@alteredworlds.com

Problems

We tried to make AlteredSaver v3 the highest quality software AlteredWorlds have yet released - and our earlier versions won 5 Star Awards from ZDNet and other awards. However, it's possible that some bugs may have crept in along the way...

Video Card Issues:
If you experience problems such as slow performance, application crashes and similar, please check the version of your display drivers. We are confident our use of OpenGL is correct: we always advise users to make sure they are using the most recent drivers offered by the manufacturer of their video card. However - updating video drivers can sometimes present difficulties. Before attempting any such sytem change, you should make sure to back up any important data, just in case. [AlteredWorlds can't be held responsible for any issues resulting from changing video drivers].

The bottom line: if there's any doubt, please use our free shareware release to test AlteredSaver on your system before buying it!

Tested Video Hardware:
Any graphics card based on an NVIDIA chip (RIVA128, TNT, TNT2, GeForce and better)
Matrox G200, G400 and newer
Kyro II (using a PowerVR chip)

ATI chips from the Rage Pro and newer are perfectly capable of running our software - depending on the OpenGl driver support. Under Linux these cards work well, but on Windows ATI's drivers can be suspect at times. Same goes for cards from 3Dfx. Please report your results to us!
